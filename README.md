# to-do-list

Steps to run the project

1. create .env file (Example mentioned)

NODE_ENV = 'development'
PORT=4000
MONGODB_URL='mongodb://127.0.0.1:27017/to-do-list'
JWT_SECRET = 'secret'
JWT_ACCESS_EXPIRATION_MINUTES=1440

2. npm install

3. npm run dev
